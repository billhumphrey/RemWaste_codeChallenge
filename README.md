# Code Challenge


# How to run this project:
1. clone repository
2. Move to the project director:
   cd remwaste
4. Download dependencies:
   npm install
6. The run:   npm run dev
7. You'll see something like this:   http://localhost:5173/
8. Follow that link to the broswer and see the page
